# Images Folder

Place your images here for the Conscious Nurturing website.

## Recommended Images

### Logo
- **Filename**: `logo.png` or `logo.svg`
- **Size**: 200x60px (or proportional)
- **Format**: PNG with transparency or SVG

### Hero/Banner Images
- **Filenames**: `hero-bg.jpg`, `banner.jpg`
- **Size**: 1920x800px
- **Format**: JPEG (optimized)

### Team Photos
- **Filenames**: `team-1.jpg`, `team-2.jpg`, etc.
- **Size**: 400x400px (square)
- **Format**: JPEG

### Service Icons
- **Filenames**: `service-1.jpg`, `service-2.jpg`, etc.
- **Size**: 600x400px
- **Format**: JPEG or PNG

### Event Photos
- **Filenames**: `event-1.jpg`, `event-2.jpg`, etc.
- **Size**: 800x600px
- **Format**: JPEG

## Image Optimization Tips

1. **Compress images** before uploading (use TinyPNG, ImageOptim, or similar)
2. **Use appropriate formats**:
   - JPEG for photos
   - PNG for graphics with transparency
   - SVG for logos and icons
3. **Resize images** to the dimensions you'll actually use
4. **Use descriptive filenames**: `mental-health-workshop.jpg` instead of `img123.jpg`

## How to Add Images

1. Place image files in this folder
2. Reference them in HTML using relative paths:
   ```html
   <img src="images/logo.png" alt="Conscious Nurturing Logo">
   ```
